# Iris Flower Classification

## Project Description

This project uses Machine Learning to classify Iris flowers into three species:

1. Setosa
2. Versicolor
3. Virginica

The model is trained using the Random Forest Classifier algorithm.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Run the Project

pip install -r requirements.txt

python main.py
